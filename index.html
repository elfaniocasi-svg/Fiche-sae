Portfolio d'apprentissage

SAÉ 1.04 — Se présenter sur Internet

Réalisé par Elfanio Casimiro · IUT Colmar, Département RT

🇬🇧 English summary

In this project, I created a personal CV website using HTML and CSS from scratch. The goal was to present myself online in a structured and professional way. I designed several web pages including a homepage, a skills section, and a contact page, all linked together with a consistent stylesheet. This project helped me understand the fundamentals of web development: semantic HTML, CSS styling, page layout, and navigation between pages.

01 · Contexte du projet

Créer un site CV en HTML & CSS

Dans le cadre de la SAÉ 1.04 « Se présenter sur Internet » à l'IUT de Colmar (département Réseaux et Télécommunications), j'ai dû concevoir et réaliser un site web personnel faisant office de CV en ligne.

Le site devait comporter plusieurs pages HTML reliées par une même feuille de style CSS, avec une navigation fonctionnelle, des ancres internes, et un rendu cohérent sur navigateur. L'outil principal utilisé pour le développement et les tests était CodePen.io.

02 · Tâches réalisées personnellement

Ce que j'ai réalisé

  • 📄
    Structure HTML des pages J'ai écrit l'ensemble du code HTML : balises sémantiques (header, nav, main, section, footer), listes, tableaux, liens et ancres.
  • 🎨
    Feuille de style CSS unique J'ai créé un fichier CSS partagé entre toutes les pages, avec mise en forme des couleurs, polices, marges et arrière-plans.
  • 🔗
    Navigation multi-pages J'ai mis en place les liens entre les différentes pages HTML du site, ainsi que des ancres pour les sauts internes.
  • 📱
    Mise en page responsive J'ai adapté les éléments visuels pour que le rendu reste correct quelle que soit la largeur de la fenêtre.
03 · Difficultés rencontrées

Les obstacles que j'ai rencontrés

Liens locaux sur CodePen

Je n'avais pas d'abonnement Pro sur CodePen, donc je ne pouvais pas tester les liens entre mes pages facilement. J'ai dû utiliser des URLs de pens séparés avec target="blank" pour contourner le problème.

Cascade CSS & conflits

Mes règles CSS se superposaient sans que je comprenne pourquoi. J'ai mis du temps à réaliser que certains sélecteurs s'annulaient entre eux, notamment pour les marges et les couleurs.

Structuration sémantique

Je ne savais pas toujours quelle balise utiliser entre article, section ou div. J'ai dû relire la doc et faire plusieurs essais avant de trouver la bonne structure.

Comportement scroll-behavior

J'avais ajouté scroll-behavior: smooth en CSS mais ça ne marchait pas sur tous les navigateurs. J'ai dû vérifier la compatibilité et adapter mon code en conséquence.
